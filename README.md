## About Me

I am **Aditya Gandhi**, a Third Year CSE Core Student and an aspiring Data Analyst. I am deeply interested in statistics, machine learning, and data visualization. I enjoy transforming messy, real-world data into structured formats and building models that help uncover hidden trends and drive decision-making.

My work sits at the intersection of **data engineering**, **statistical analysis**, and **business intelligence**. I like building data pipelines, optimizing SQL queries, and creating intuitive dashboards that explain complex metrics clearly.

## Personal Note

I am currently focused on building a strong portfolio of data-driven projects: exploratory data analysis, predictive modeling, and interactive dashboards. I believe that data is only as good as the story it tells, and I strive to make my analyses readable, reproducible, and impactful.

## Current Focus

- **Data Analytics** - Mastering Python libraries (Pandas, NumPy) and advanced SQL for data manipulation.
- **Machine Learning** - Exploring predictive modeling, classification, and clustering algorithms using Scikit-Learn.
- **Data Visualization** - Building interactive dashboards using Power BI, Tableau, and Matplotlib/Seaborn.
- **Database Management** - Designing efficient schemas and optimizing query performance in PostgreSQL and MySQL.
